# Landing profesional inicial

Este es mi primer proyecto dentro de mi plan de aprendizaje para convertirme en desarrollador frontend.

La página está construida con HTML y CSS puro. El objetivo del proyecto es practicar estructura semántica, diseño responsive, componentes visuales reutilizables y buenas prácticas básicas de organización.

## Tecnologías utilizadas

- HTML5
- CSS3
- Git
- GitHub

## Qué he practicado

- Estructura básica de una página HTML
- Uso de etiquetas semánticas como `header`, `main`, `section`, `article`, `nav` y `footer`
- Selectores CSS
- Modelo de caja: `margin`, `padding`, `border`
- Flexbox
- Diseño responsive con media queries
- Formularios HTML
- Navegación interna con `id` y enlaces `href="#seccion"`
- Efectos visuales con `hover`, `transition` y `transform`

## Secciones de la página

- Inicio
- Proyecto destacado
- Tarjetas de habilidades/proyectos
- Formulario de contacto
- Footer
